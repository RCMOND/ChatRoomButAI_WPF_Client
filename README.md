## ChatRoomWpf - 桌面聊天客户端

基于 WPF (.NET 8) 的桌面聊天客户端，与 Web 端功能同步，支持独立登录弹窗、音乐播放、文件上传、头像更换等。

## 技术栈
- .NET 8 WPF
- SignalR Client
- MediaPlayer (音频播放)

## 功能特性
- 独立登录/注册弹窗，记住登录状态（本地凭据）
- 实时聊天（文字、图片、文件）
- 全局音乐同步（播放、暂停、切歌、循环模式）
- 播放列表查看与点播
- 在线用户列表及 @提及
- 头像点击上传并实时同步
- 系统公告显示
- 文件上传进度条
- 退出自动弹出登录窗口

## 环境要求
- Windows 10/11 或 Windows Server 2016+
- .NET 8 Desktop Runtime (如需独立运行可发布自包含版本)
